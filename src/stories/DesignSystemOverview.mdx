# Echo Design System Overview

The Echo Design System is a comprehensive, scalable design foundation built with React, TypeScript, and modern CSS practices. This system provides a complete set of design tokens, components, and patterns for building consistent, accessible user interfaces.

## üé® Color System

### Comprehensive Color Palette
We've implemented a professional-grade color system with **67 design tokens** organized into logical palettes:

#### Primary Palettes
- **Blue Palette**: Traditional brand colors (50-900 scale)
- **Sky Blue Palette**: Interactive states and focus indicators (50-950 scale)
- **Yellow Palette**: Attention, highlights, and warnings (50-950 scale)
- **Seafoam Green Palette**: Nature, growth, and positive actions (50-950 scale)

#### Neutral & Base Colors
- **Extended Base Grays**: Professional neutral palette (50-950 scale)
- **Pure Base Colors**: Pure white, black, and gray for high contrast

#### Status Colors
- **Extended Status Palettes**: Complete Red, Orange, and Green scales for comprehensive feedback states
- **Echo-Specific Status Colors**: Branded validation colors for forms and interactions

### Design Token Architecture
All colors follow consistent naming conventions:
```css
--primary-[color]-[weight]     /* Primary palettes */
--base-gray-[weight]           /* Neutral grays */
--status-[color]-[weight]      /* Status feedback */
```

## üöÄ Getting Started

### Installation
The design system is ready to use with all dependencies configured:
```bash
npm install
npm run storybook
```

### Basic Usage
```tsx
import { Input, Textarea, Tag } from './components';

// Basic input with validation
<Input 
  label="Email"
  state="error"
  message="Please enter a valid email"
  size="default"
/>

// Textarea with enhanced features
<Textarea
  label="Description"
  state="focus"
  size="large"
  rows={4}
/>

// Interactive tags
<Tag
  label="Design System"
  variant="primary"
  size="default"
  showClose={true}
/>
```
