## Migration Guides

### Upgrading from 1.x to 2.x

#### Color Token Updates
The color system has been significantly expanded. Existing tokens remain compatible:

```css
/* ‚úÖ Still works */
--primary-blue-blue-500
--neutral-gray-gray-300
--semantic-success

/* ‚ú® New options available */
--primary-sky-blue-500
--primary-yellow-500
--primary-seafoam-500
```

#### Component Props
All existing component props remain the same. New features are additive:

```tsx
// ‚úÖ Existing usage still works
<Input label="Email" state="error" />

// ‚ú® New features available
<Input 
  label="Email" 
  state="focus" 
  size="large" 
  showLeadingIcon={true} 
/>
```

#### Performance
Components now use CSS containment for better performance. No breaking changes to existing functionality.
