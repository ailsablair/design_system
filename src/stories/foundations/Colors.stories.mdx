## Usage Guidelines

### Accessibility
- All color combinations meet WCAG 2.1 AA contrast requirements
- Never rely on color alone to convey information
- Test colors with common color vision deficiencies

### Implementation
Colors are available as CSS custom properties:
```css
.element {
  background-color: var(--primary-blue-blue-500);
  color: var(--base-white);
}
```

### Best Practices
- Use primary colors sparingly for maximum impact
- Leverage neutral grays for most interface elements
- Apply semantic colors consistently for their intended purpose
- Maintain sufficient contrast ratios for readability
